# 聖剣3 属性ありの敵の技

## 計測方法
+ 神獣リングの交換でダメージが半減すれば、属性ありとみなす
+ 8個全てダメージが同じなら、無属性とみなす

## 前提
+ 計測結果は目視判定のため、実際とは異なる場合があります
+ 「使用者」欄には、実際に計測した敵のみを記載（未計測の敵は未記載）
+ 「ダウン有無」「ヒット数」などは、使用者によって異なる場合があります（記載内容は暫定）
